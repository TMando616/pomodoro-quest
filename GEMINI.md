# GEMINI.md

このファイルは、**pomodoro-quest** プロジェクトで Gemini CLI が作業する際の指示とコンテキストを提供します。

## プロジェクト概要
- **名称:** pomodoro-quest
- **スタック:** Next.js (App Router), TypeScript, Tailwind CSS v4, Lucide React.
- **目的:** RPG/クエスト要素を組み合わせた没入型ポモドーロタイマーアプリ。

## 実装済み機能
1.  **クエストタイマー:** 5分〜60分（5分刻み）で時間を設定可能。
2.  **動的経験値システム:** 集中時間と難易度に応じたEXP獲得とレベルアップ。
3.  **冒険者ギルド (認証):** ユーザー登録・ログイン機能。最初の一人目は自動で管理者権限を付与。
4.  **マルチ言語対応 (i18n):** 日本語と英語の即時切り替え。カスタムイベントによるタブ内全コンポーネントの同期。
5.  **没入型スタート画面:** ゲームタイトルのような導入。言語選択と認証を統合。一度開始すると永続化（スキップ）可能。
6.  **マルチテーマ (Affinity):** Shadow（ダーク）/ Radiance（ライト）各5色の動的UI。テーマ情報の永続化。
7.  **ステータスHUD:** 名前、称号、レベル、経験値バー。ハイドレーションエラー対策済み。
8.  **冒険者ガイド (ヘルプ):** 基本操作、主要システム、FAQを網羅したガイドページ (`/help`)。
9.  **称号システム:** 条件達成でアンロックされる称号の装備機能。
10. **ボス討伐モード:** 一時停止不可・EXP2倍・失敗ペナルティありのハードコアモード。
11. **データ永続化:** LocalStorageによる全データの保存。マウント後の非同期読み込みによるHydration一致。

## 技術的特徴と設計指針

### 1. ハイドレーションエラー (Hydration Mismatch) 対策
Next.js (SSR) 環境で `localStorage` を安全に使用するため、以下のパターンを徹底しています。
- **原則:** `useState` の初期値で `localStorage` を参照しない（常にサーバーと一致するデフォルト値を使用）。
- **実装:** `useEffect` 内でマウントを確認 (`setIsMounted(true)`) してからデータを読み込み、表示を更新する。
- **UI保護:** ユーザー状態に依存する要素（リンクの `href` や称号表示など）は、`isMounted` が `true` になるまでデフォルト値を維持する。

### 2. 設定の即時同期 (Cross-Component Sync)
設定画面で変更した言語やテーマを、ページ遷移なしでナビゲーションバー等の別コンポーネントに反映させるため、ブラウザの `CustomEvent` を利用しています。
- `useSettings` フックが変更を検知すると `pq-settings-updated` イベントを発火。
- 同一タブ内の他の `useSettings` インスタンスがイベントを購読し、自身の状態を更新することで即時反映を実現。

### 3. 多言語対応 (i18n)
- `src/constants/translations.ts` にすべての文言を集約。
- `src/hooks/useTranslation.ts` を通じて、現在の設定言語に基づいた翻訳オブジェクトを提供。

## ディレクトリ構成
- `src/app`: ページエントリーポイント、グローバルスタイル、各ルート (`/profile`, `/guild`, `/help` 等)。
- `src/components/layout`: ナビゲーション、プロバイダー、言語管理などの基盤コンポーネント。
- `src/components/ui`: 分割されたUIコンポーネント。
- `src/hooks`: ビジネスロジックをカプセル化したカスタムフック。
- `src/constants`: テーマデータ、称号定義、翻訳辞書などの定数。

## ルールと規約
- **コンポーネント:** UIセクションごとにコンポーネントを細かく分割し、再利用性を高める。
- **検証:** 変更確定前に必ず `npm run lint` を実行し、Reactのルール（特に不純な関数の排除やカスケードレンダリングの回避）を遵守する。
- **コミット:** Conventional Commits に従い、1機能1コミットを原則とする。
