# Pomodoro Quest ⚔️

RPGの世界観とポモドーロタイマーを融合させた、集中力を高めるための「冒険」ツールです。
作業時間を「クエスト」としてクリアし、経験値を稼いで伝説の冒険者を目指しましょう。

## 🛡️ 主な機能

### 🧭 クエストと休息
- **自在な時間設定:** 5分から60分まで、作業内容に合わせてクエスト時間を調整。
- **休息システム:** クエスト完了後は宿屋（Rest）で休息。Short / Long Rest を選択可能。
- **クエスト命名:** 「宿敵のバグを討伐」「魔導書（技術書）の読破」など、自由に名前を付けて開始。

### ⚔️ ゲーム要素 (RPG Elements)
- **レベルと経験値:** 集中した時間に応じてEXPを獲得。1000 EXPごとにレベルアップ。
- **称号システム:** 「集中の騎士」「時の賢者」など、特定の条件を満たすと称号が解禁。
- **ボス討伐モード:** 逃げ場なし（一時停止不可）の決戦モード。獲得EXPは2倍だが、逃走にはペナルティも。
- **冒険の記録:** 過去のクエスト達成履歴をいつでも「冒険の記録」から振り返り可能。

### 🎨 没入感を高めるUI & システム
- **マルチ言語対応 (i18n):** 日本語と英語に対応。設定からいつでも即座に切り替え可能。
- **没入型スタート画面:** アプリ起動時のタイトル画面で、ログイン・登録・ゲスト利用を選択。
- **マルチテーマ (Affinity):** 5つの属性と、Shadow（ダーク）/ Radiance（ライト）モードを自由に組み合わせ。
- **データ永続化:** `localStorage` を活用し、ブラウザを閉じても進行状況や設定を保持。

## 🚀 技術スタック

- **Framework:** Next.js 15 (App Router)
- **Styling:** Tailwind CSS v4
- **Icons:** Lucide React
- **State Management:** React Hooks + Custom Events (Settings Sync)
- **Language:** TypeScript

## 🛠️ 遊び方

1.  **冒険の始まり:** スタート画面で言語を選択し、「新規登録」または「ゲスト」として開始。
2.  **クエスト準備:** メイン画面でクエスト名を入力し、時間をセット。
3.  **クエスト開始:** プレイボタンを押して集中開始。ボスモード中は一時停止不可！
4.  **リザルト:** 完了時にEXPを獲得。称号を集めて自分だけのプロフィールを完成させましょう。

## 👨‍💻 開発の始め方

### 環境構築
```bash
# リポジトリのクローン後
npm install
```

### 開発サーバーの起動
```bash
npm run dev
```
`http://localhost:3000` でアプリが起動します。

### コードの検証
```bash
npm run lint
```
プルリクエスト前には必ずリントを実行し、ハイドレーションエラーやReactのルールに違反がないか確認してください。
