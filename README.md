# Pomodoro Quest ⚔️

RPGの世界観とクエスト要素を組み合わせた、没入感のあるポモドーロタイマーアプリです。
集中時間を「クエスト」としてこなし、冒険者のレベルを上げていきましょう。

## 🛡️ 主な機能

- **クエストタイマー:** 集中したい時間（5分〜60分）を選択してクエスト開始。
- **冒険者レベルシステム:** クエストを完了すると経験値（EXP）を獲得し、レベルアップ。
- **冒険者ログ (Auth):** ギルドに登録（ユーザー登録）して、自分の成長を記録。
- **マルチテーマ (Affinity):** 
  - **Shadow:** 深淵を感じさせるダークテーマ（Emerald, Ruby, Sapphire, Amber, Amethyst）。
  - **Radiance:** 光り輝くライトテーマ。
- **レスポンシブデザイン:** モバイルからデスクトップまで、あらゆる端末での冒険に対応。

## 🚀 技術スタック

- **Framework:** Next.js 15 (App Router)
- **Styling:** Tailwind CSS v4 (Modern CSS variables approach)
- **Icons:** Lucide React
- **Language:** TypeScript
- **State Management:** React Hooks (In-memory mock database)

## 🛠️ 開発の始め方

1.  **依存関係のインストール:**
    ```bash
    npm install
    ```

2.  **開発サーバーの起動:**
    ```bash
    npm run dev
    ```

3.  **ブラウザで確認:**
    `http://localhost:3000` を開きます。
