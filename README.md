# Pomodoro Quest ⚔️

RPGの世界観とポモドーロタイマーを融合させた、集中力を高めるための「冒険」ツールです。
作業時間を「クエスト」としてクリアし、経験値を稼いで伝説の冒険者を目指しましょう。

## 🛡️ 主な機能

### 🧭 クエストと休息
- **自在な時間設定:** 5分から60分まで、作業内容に合わせてクエスト時間を調整。
- **休息システム:** クエスト完了後は宿屋（Rest）で休息。Short / Long Rest を選択可能。
- **クエスト命名:** 「宿敵のバグを討伐」「魔導書（技術書）の読破」など、自由に名前を付けて開始。

### ⚔️ ゲーム要素 (RPG Elements)
- **レベルと経験値:** 集中した時間に応じてEXPを獲得。1000 EXPごとにレベルアップ。
- **称号システム:** 「Focus Knight」「Time Sage」など、特定の条件を満たすと称号が解禁。
- **ボス討伐モード:** 逃げ場なし（一時停止不可）の決戦モード。獲得EXPは2倍だが、逃走（リセット）にはペナルティも。
- **冒険の記録:** 過去のクエスト達成履歴をいつでも「Adventure Log」から振り返り可能。

### 🎨 没入感を高めるUI
- **マルチテーマ (Affinity):** 5つの属性（Emerald, Ruby, Sapphire, Amber, Amethyst）と、Shadow/Radianceの明暗モードから選択。
- **動的エフェクト:** テーマカラーに合わせた光彩（Glow）効果や、ボス戦時の緊迫した赤色UI。
- **データ永続化:** ブラウザを閉じても、あなたのレベルや称号、履歴は大切に保存されます。

## 🚀 技術スタック

- **Framework:** Next.js 15 (App Router)
- **Styling:** Tailwind CSS v4
- **Icons:** Lucide React
- **Persistence:** LocalStorage API
- **Language:** TypeScript

## 🛠️ 遊び方（開発の始め方）

1.  **ギルドに登録:** 「Adventurer Log」から自分の名前を登録。
2.  **クエスト準備:** クエスト名を入力し、時間をセット。必要なら「ボス戦」をONに。
3.  **クエスト開始:** プレイボタンを押して集中開始。一時停止は許されません（ボス戦時）。
4.  **リザルト:** 完了時にEXPを獲得。レベルアップを目指しましょう！

```bash
npm install
npm run dev
```
